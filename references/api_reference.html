Parameters

  • model (torch.nn.Module) – The global model for training.

  • server_optimizer (torch.optim.Optimizer or str) – Pytorch optimizer for server-side optimization. Currently, the str type only supports SGD

  • client_optimizer (torch.optim.Optimizer or str) – Pytorch optimizer for client-side optimization. Currently, the str type only supports SGD

  • loss (str) – A Pytorch Loss function. See https://pytorch.org/docs/stable/nn.html#loss-functions. Currently, the str type only supports SGD

  • global_rounds (int, optional) – Number of communication rounds in total.

  • local_steps (int, optional) – Number of local steps of each global round.
