class aggregators.krum.Krum(num_clients=20, num_byzantine=5)[source]

Bases: aggregators.mean._BaseAggregator

A robust aggregator from paper “Machine Learning with Adversaries: Byzantine Tolerant Gradient Descent”

Given a collection of vectors, Krum strives to find one of the vector that is closest to another \(K-M-2\) ones with respect to squared Euclidean distance, which can be expressed by:

\[Krum := \{{\Delta}_i | i = \arg\min_{i \in [K]} \sum_{i \rightarrow j} \lVert {\Delta}_i - {\Delta}_j \rVert^2 \}\]

where \(i \rightarrow j\) is the indices of the \(K-M-2\) nearest neighbours of \({\Delta}_i\) measured by squared Euclidean distance, \(K\) is the number of clients in total, and \(M\) is the number of Byzantine clients.

class aggregators.geomed.Geomed(maxiter: Optional[int] = 100, eps: Optional[float] = 1e-06, ftol: Optional[float] = 1e-10)[source]

Bases: aggregators.mean._BaseAggregator

A robust aggregator from paper “Distributed Statistical Machine Learning in Adversarial Settings: Byzantine Gradient Descent”

GeoMed aims to find a vector that minimizes the sum of its Euclidean distances to all the update vectors:

\[GeoMed := \arg\min_{\boldsymbol{z}} \sum_{k \in [K]} \lVert \boldsymbol{z} - {\Delta}_i \rVert.\]

There is no closed-form solution to the GeoMed problem. It is approximately solved using Weiszfeld’s algorithm in this implementation to.

Parameters

  • maxiter – Maximum number of Weiszfeld iterations. Default 100

  • eps – Smallest allowed value of denominator, to avoid divide by zero. Equivalently, this is a smoothing parameter. Default 1e-6.

  • ftol – If objective value does not improve by at least this ftol fraction, terminate the algorithm. Default 1e-10.

class aggregators.trimmedmean.Trimmedmean(num_byzantine=5)[source]

Bases: aggregators.mean._BaseAggregator

A robust aggregator from paper Byzantine-robust distributed learning: Towards optimal statistical rates

It computes the coordinate-wise trimmed average of the model updates, which can be expressed by:

\[trmean := TrimmedMean( \{ \Delta_k : k \in [K] \} ),\]

where the \(i\)-th coordinate \(trmean_i = \frac{1}{(1-2\beta)m} \sum_{x \in U_k}x\), and \(U_k\) is a subset obtained by removing the largest and smallest \(\beta\) fraction of its elements.
