class client.BladesClient(id: Optional[str] = None, device: Optional[str] = 'cpu')[source]#

Bases: object

Base class for all clients.

Note

Your honest clients should also subclass this class.

Parameters

  • id (str) – a unique id of the client.

  • device (str) – target device if specified, all parameters will be copied to that device

id() → str[source]#

Returns the unique id of the client.

Example

>>> from blades.client import BladesClient
>>> client = BladesClient(id='1')
>>> client.id()
'1'

class client.ByzantineClient(id: Optional[str] = None, device: Optional[str] = 'cpu')[source]#

Bases: client.BladesClient

Base class for Byzantine input.

Note

Your Byzantine input should also subclass this class, and overwrite local_training and omniscient_callback to customize your attack.

omniscient_callback(simulator)[source]#

A method that will be registered by the simulator and execute after each communication round. It allows a Byzantine client has full knowledge of the training system, e.g., updates from all input. Your Byzantine client can overwrite this method to access information from the server and other input.

Parameters

simulator (Simulator) – The running simulator.
