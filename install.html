1. Installation

1.1. Single Machine Simulation

To use Blades on a single machine, you only need to install the simulator using pip:

>>> pip install blades

1.2. Ray Cluster Simulation

Borrowing the power of Ray, Blades can be easily adapted to clusters without changing a single line of the code.

See also

See how to deploy Ray cluster according to the official guide. Make sure that both Python and Ray on all machines has the same version.
